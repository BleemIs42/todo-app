# todo-app

## Setup
```sh
# install node
brew install node

# init react project 
yes | npm create vite todo-app -- --template react-ts

# install dependence
cd todo-app
npm i

# run dev to see page
npm run dev
```

## Lesson 1
```sh
# to see task
git switch todo-1

# to see expect result
git switch todo-1-done
```
